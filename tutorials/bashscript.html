how to make a bash script,

today i will teach you now to make a Bash script, to Install Apache2


so first of all you are going to want to make sure you are root, Or have sudo access
Then run This command:
sudo touch run.sh
so that makes you a File called run.sh, then you want to start editing it using NANO with this command:
sudo nano run.sh
then a text editor will pop up Then type:
sudo apt install apache2
then press control S and Control X, that will return you to the terminal, then run:
bash run.sh


then it should work now!
